PROJECT NAME:
PlantHealth AI – Smart Plant Disease Detection System


PROJECT OVERVIEW:
PlantHealth AI is an artificial intelligence-based plant disease detection system developed using deep learning techniques.

The system takes a plant leaf image as input, analyzes it using a trained deep learning model, and predicts the plant disease along with confidence score.


PROBLEM STATEMENT:
Farmers and plant growers often face difficulty in identifying plant diseases at an early stage through manual inspection.

This project aims to automate disease detection using image classification.


OBJECTIVE:
To build an intelligent system capable of:
1. Detecting plant diseases from leaf images
2. Classifying healthy and diseased leaves
3. Providing confidence-based predictions


DATASET:
Dataset Name: PlantVillage Dataset

Number of Classes: 38

Dataset contains multiple plant species with healthy and diseased leaf images.


MODELS USED:
1. Custom CNN
2. Transfer Learning using MobileNetV2


TECHNOLOGIES USED:
Python
TensorFlow
NumPy
Matplotlib
Pillow
Streamlit
VS Code


PROJECT FEATURES:
1. Plant disease detection
2. Confidence score prediction
3. User-friendly web interface
4. Real-time image upload and prediction


PROJECT RESULTS:
The transfer learning model achieved high classification performance and successfully predicted multiple plant diseases on unseen images.


HOW TO RUN THE PROJECT:

pip install -r requirements.txt

Run Streamlit application
